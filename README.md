# Zenith - Real Estate Website

**Course:** Internet Technologies
**Faculty:** CST, SEE University  
**Semester:** 2nd Semester (2022/2023)

## About

Zenith is my first frontend project during my university studies. It is a real estate website that helps users find their dream home with an intuitive interface and modern design.

## Features

- **Property Search**: Advanced search functionality with filters for property type and price range
- **Featured Listings**: Showcases top-rated properties with detailed information
- **Property Details**: View property specifications including size, bedrooms, and bathrooms
- **Video Integration**: Interactive video player in the about section
- **Newsletter Subscription**: Stay updated with the latest property listings
- **Responsive Design**: Fully responsive layout that works on all devices
- **Favorite Properties**: Add properties to your favorites list

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Ionicons (for icons)
- Google Fonts (League Spartan)

## How to Run

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! No build process required.
